#Graficador Matemático (InfinityMath)
### Plataforma de Exploración Matemática Visual

InfinityMath es una aplicación desarrollada en Java orientada al aprendizaje visual e interactivo de conceptos matemáticos y de geometría analítica.

El proyecto nació inicialmente como un graficador matemático, pero evolucionó hacia una herramienta educativa capaz de representar visualmente funciones y objetos geométricos, permitiendo al estudiante explorar relaciones matemáticas de forma dinámica y comprensible.

---

# Características actuales

## Módulo de Funciones
- Graficación de funciones cuadráticas
- Representación en plano cartesiano
- Detección de raíces
- Visualización de máximos y mínimos
- Sistema de ejes dinámico

## Módulo de Geometría Analítica
- Representación de círculos
- Representación de rectas
- Detección exacta de intersecciones
- Clasificación automática:
  - Recta secante
  - Recta tangente
  - Recta exterior
- Visualización de puntos de intersección

## Interfaz
- Sistema modular por pestañas
- Separación entre álgebra y geometría
- Interfaz educativa orientada al aprendizaje visual
- Arquitectura escalable para futuras mejoras

# Objetivo del proyecto

El propósito principal de InfinityMath es facilitar la comprensión matemática mediante representación visual e interacción gráfica.

La plataforma busca servir como apoyo educativo para:
- estudiantes
- docentes
- tutorías académicas
- exploración matemática visual
